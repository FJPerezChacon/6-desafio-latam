Este proyecto de diseño web consistió en la creación de la página oficial para "Villa Mascotas", una tienda especializada en productos y servicios para mascotas. El objetivo principal era crear una plataforma atractiva, informativa y fácil de usar que transmitiera la esencia del negocio

"Iguana Page" es una página web diseñada para ser un recurso informativo completo para todo aquel que tenga interés en las iguanas. Su diseño minimalista, con colores oscuros y llamativos, crea una atmósfera elegante y moderna que invita a la exploración.

"Coppon" es una página web que te permite encontrar cupones de descuento para una amplia variedad de empresas. Su diseño minimalista y colorido, con tonos pastel y un sistema de tarjetas de Bootstrap, crea una experiencia de usuario amigable e intuitiva.

Un repositorio en GitHub es una excelente opción para ti. Te permite compartir los proyectos, ideas y conocimientos de una persona con una amplia audiencia, desde amigos y familiares hasta posibles empleadores y clientes.
